# Bookstore-app
This project is a full-stack bookstore web application built with the MERN stack (MongoDB, Express.js, React.js, and Node.js) that allows users to browse free books and access paid books after authentication. The frontend uses React, React Router, Context API, Tailwind CSS, and DaisyUI to provide a responsive UI with features such as a slider for free books, protected routes, login/signup modals, toast notifications, and a dark/light mode toggle stored in local storage. The backend exposes RESTful APIs using Express and MongoDB with Mongoose, handling book data retrieval and secure user authentication with bcrypt password hashing.

To run this project locally, you must configure environment variables for the backend (MongoDB connection string and server port) and install dependencies for both the frontend and backend before starting the development servers.
